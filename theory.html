Общая химия

Строение атома

"Атом - это как мини-вселенная: в центре ядро (протоны и нейтроны), а вокруг летают электроны, как планеты!"
- Георгий объясняет просто

Основные понятия:

  • Атом состоит из ядра (протоны + нейтроны) и электронной оболочки
  • Заряд ядра равен порядковому номеру элемента в таблице Менделеева
  • Число электронов = числу протонов (в нейтральном атоме)

Формулы и законы:

A = Z + N

Где:
A - массовое число
Z - число протонов (зарядовое число)
N - число нейтронов

Периодический закон

"Периодическая таблица - это как паспортный стол для элементов: у каждого есть свой номер и характерные свойства!"
- Георгий о таблице Менделеева

Основные понятия:

  • Свойства элементов периодически зависят от заряда их атомных ядер
  • Горизонтальные ряды - периоды (7 штук)
  • Вертикальные столбцы - группы (8 главных)

Закономерности:

В периодах (слева направо):
  • ↑ Заряд ядра
  • ↓ Атомный радиус
  • ↑ Окислительные свойства
В группах (сверху вниз):
  • ↑ Атомный радиус
  • ↑ Восстановительные свойства
  • ↓ Электроотрицательность

Химические реакции

Типы химических реакций

1. Реакции соединения

2Mg + O2 → 2MgO

Из нескольких простых или сложных веществ образуется одно сложное

2. Реакции разложения

2H2O → 2H2 + O2

Из одного сложного вещества образуется несколько простых или сложных веществ

3. Реакции замещения

Zn + 2HCl → ZnCl2 + H2

Атомы простого вещества замещают атомы одного из элементов в сложном веществе

4. Реакции обмена

AgNO3 + NaCl → AgCl + NaNO3

Сложные вещества обмениваются своими составными частями

Определи тип реакции:
CaCO3 → CaO + CO2

Органическая химия

Классы органических соединений

Углеводороды

Метан (CH4)
  • Алканы (предельные) - CnH2n+2
  • Алкены (двойная связь) - CnH2n
  • Алкины (тройная связь) - CnH2n-2

Кислородсодержащие

Этанол (C2H5OH)
  • Спирты - R-OH
  • Карбоновые кислоты - R-COOH
  • Альдегиды и кетоны

Азотсодержащие

Метиламин (CH3NH2)
  • Амины - R-NH2
  • Аминокислоты
  • Нитросоединения
Попрактикуйся в номенклатуре:
CH3-CH2-CH3
