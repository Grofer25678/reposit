При взаимодействии 16,2 г алюминия с серой образовалось 30,6 г сульфида алюминия. Определите массовую долю выхода продукта реакции от теоретически возможного.

Ваш ответ:

Решение:

  1. Записываем уравнение реакции: 2Al + 3S → Al₂S₃
  2. Находим количество вещества алюминия: n(Al) = m/M = 16,2/27 = 0,6 моль
  3. По уравнению реакции: n(Al₂S₃) = 1/2 n(Al) = 0,3 моль
  4. Теоретическая масса Al₂S₃: m(теор) = n*M = 0,3*150 = 45 г
  5. Выход продукта: η = m(практ)/m(теор)*100% = 30,6/45*100% = 68%
Ответ: 68%
